# Car Buy Website

A responsive car buying website with a Node.js backend and static frontend. The website showcases popular Indian car models, prices in INR, brand logos, and complete specifications.

## Features

- Responsive design for desktop and mobile
- Backend API serving car details
- Search and filter by brand and fuel type
- Indian pricing and car images
- Easy local setup

## Setup

1. Open a terminal in `d:\Mini-Project`
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:4000` in your browser

## Project structure

- `server.js`: Express server and API endpoint
- `data/cars.json`: Car catalog data
- `public/index.html`: Frontend markup
- `public/app.js`: Frontend application logic
- `public/styles.css`: Responsive styling

## Notes

The site uses remote images for brand cars so it loads quickly and can display real photo content.
